# ncov-koa

武汉新型冠状病毒疫情数据
数据来自[丁香园](https://ncov.dxy.cn/ncovh5/view/pneumonia)（只包含中国城市数据）

### development

```
yarn dev
```

### build

```
yarn build
```

### 路由

`/area` 获取地区数据

`/dash` 获取全国「确诊」、「疑似」、「死亡」、「治愈」总人数

### 前端数据展示

[ncov-vue](https://github.com/yoorn/ncov-vue)
